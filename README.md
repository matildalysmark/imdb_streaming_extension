# IMDb streaming extension

A browser extension that adds the IMDb rating to titles on streaming services.

<img width="250" alt="Netflix IMDb Rating Screenshot" src="https://github.com/user-attachments/assets/ed517a6e-7811-4df4-8a74-5bb2f1895502" />

### Status
* **Currently compatible with:** Netflix
* **Tested in:** Brave, Microsoft Edge
  
### Installation
1.  Download the source code from this repo
2.  Open your browser's extension manager
3.  Enable **Developer Mode**
4.  Choose the option **Load unpacked**
5.  Select the folder with the source code
